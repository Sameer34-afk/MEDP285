What is HTML?

HTML (HyperText Markup Language) is the coding language used to structure and format the content of a web page. It is basically the fundamental building blocks of the web.

What is CSS?

CSS (Cascading Style Sheets) is basically the language that's used to show how documents are laid out. This is how you customize your website and make it unique.

What is JavaScript?

JavaScript is a programming language that enables interactivity on a webpage. It allows you to get creative and fun on your website.
